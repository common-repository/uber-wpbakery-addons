== Installation ==

1. Go to plugins in your dashboard menu and select 'Add New'.
2. Search for 'Uber WPBakery Addons', install and activate it.
3. Create a new page or post, and edit it with WPBakery.
4. Look for the tab called "Uber Addons" in your WPBakery blocks list.
5. Start adding blocks to your page. More info can also be found on the screenshots bellow.
